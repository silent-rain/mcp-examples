# 环境搭建文档

## 系统环境

- 系统版本: Arch Linux x86_64
- Node版本: v24.4.0
- Pnpm版本: 10.13.1

## 搭建项目

### 创建项目

```shell
pnpm create vite

# or

pnpm create vite ts-mcp-server --template vue-ts
```

### 添加 MCP 库

```shell
pnpm add @modelcontextprotocol/sdk
```

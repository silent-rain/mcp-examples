# 环境搭建文档

## 系统环境

- 系统版本: Arch Linux x86_64
- Python版本: 3.13.1
- Rust版本: 1.88.0
- Node版本: v23.7.0
- Pnpm版本: 9.12.3
- UV版本: 0.7.18

## UV 安装

```shell
curl -LsSf https://astral.sh/uv/install.sh | sh
# or
pipx install uv
```
